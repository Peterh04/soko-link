Installation & Usage ğŸ’»

Clone the repository:

git clone https://github.com/Peterh04/soko-link-backend.git


Install dependencies for frontend and backend:

cd frontend
npm install
cd ../backend
npm install


Configure your .env files with required credentials (PostgreSQL URL, Cloudinary, M-Pesa API keys, etc.)

Start the development servers:

cd backend
npm run dev
cd ../frontend
npm run dev


Open http://localhost:3000 in your browser to view the app locally
