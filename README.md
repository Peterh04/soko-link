Soko-Link

Soko-Link is a full-stack Kenyan online marketplace that enables seamless buying and selling. Sellers can list products with images, buyers can browse and purchase via M-Pesa, and users can communicate in real-time through an integrated chat system powered by WebSockets.

Key Features

Dynamic Product Listings: Easily add, update, and display products.

Real-Time Chat: Instant communication between buyers and sellers for a more engaging marketplace experience.

Secure Payments: Smooth transactions powered by M-Pesa Daraja API.

Cloud Image Handling: Product images uploaded and served reliably via Cloudinary.

Responsive UI: Clean, mobile-friendly interface optimized for usability.

Backend & Database: Robust Node.js + Express backend with PostgreSQL for scalable data management.

Technologies Used

Frontend: React.js, HTML, CSS

Backend: Node.js, Express.js (GitHub Repo
)

Database: PostgreSQL

Payments: M-Pesa Daraja API

Real-Time Communication: WebSockets

Image Storage: Cloudinary

Hosting: Render
